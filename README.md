# 📊 E-commerce Analytics Dashboard (AI Powered)

## 🚀 Overview

This project is an end-to-end e-commerce analytics system built using Snowflake, SQL, Python, and Streamlit. It provides business insights and includes an AI-powered natural language query feature using Google Gemini.

## 🔧 Tech Stack

* Snowflake (Data Warehouse)
* SQL (Data Analysis)
* Python (Backend)
* Streamlit (Dashboard)
* Plotly (Visualization)
* Google Gemini API (AI)

## 📊 Features

* Revenue, Customers, Orders KPIs
* Monthly Revenue Trends
* Customer Retention & Churn Analysis
* Cohort Retention Analysis
* Top Products Analysis
* Revenue by State
* New vs Repeat Customers
* AI-powered "Ask Data" feature

## 🤖 AI Feature

Users can ask questions like:

* "Top 5 products by revenue"
* "Monthly revenue trend"

The system converts natural language into SQL and fetches results from Snowflake.

## ⚙️ Setup

1. Clone repo
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Add `.env` file:

   ```
   GEMINI_API_KEY=your_api_key
   ```
4. Run:

   ```
   streamlit run app.py
   ```
